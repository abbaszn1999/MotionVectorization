Motion Vectorization

Editing Motion Graphics Video via Motion Vectorization and Program Transformation

About this project:
This system converts videos into animated SVG graphics by analyzing motion patterns, extracting shapes, and creating smooth vector animations. It uses RAFT optical flow estimation and advanced shape tracking algorithms.

Project Information

Main Components:

  • RAFT: Optical flow estimation model
  • Motion Vectorization: Shape extraction and tracking
  • SVG Utils: Vector graphic generation
  • GUI Tool: Manual correction interface

Pipeline: Video → Flow Analysis → Shape Clustering → Tracking → Optimization → SVG
